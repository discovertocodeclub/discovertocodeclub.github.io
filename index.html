What Is HTML

HTML is a computer language devised to allow website creation. These websites can then be viewed by anyone else connected to the Internet. It is relatively easy to learn, with the basics being accessible to most people in one sitting; and quite powerful in what it allows you to create. It is constantly undergoing revision and evolution to meet the demands and requirements of the growing Internet audience.

What is CSS?

CSS stands for Cascading Style Sheets. A.K.A. the designer. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. Stylesheets are stored in CSS files.

What Is Javascript

An computer programming language commonly used to create interactive effects within web browsers or its conten. It is often used in websites, programs and, games.
